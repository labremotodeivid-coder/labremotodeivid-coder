# 👨🏻‍🔬 Astolfinho Smith

**`Estudante de Física | Ajudante no Lab Remoto`**

Também compartilho conteúdo educativo pelo YouTube e pelo Instagram, com o perfil chamado **"Lab Remoto"**.
